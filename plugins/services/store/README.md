# Store Service Plugin

应用全局状态管理服务插件。

## 功能概述

Store Service 插件为应用提供全局状态管理能力，主要负责：

- **用户认证状态管理**：管理用户登录状态、用户信息等认证相关状态
- **应用加载状态管理**：管理应用全局加载状态
- **自动登录支持**：支持记住用户凭证并在应用启动时自动登录
- **状态持久化**：与后端系统集成，支持状态的持久化存储

## 主要功能

### 用户认证管理

- 维护当前登录用户的信息
- 跟踪用户登录状态
- 支持用户登录和登出操作
- 支持应用启动时的自动登录

### 应用状态管理

- 管理应用全局加载状态
- 提供统一的状态访问接口
- 支持状态在组件间的共享

## 系统集成

Store Service 与以下系统集成：

- **路由系统**：根据登录状态自动切换应用布局和授权布局
- **后端服务**：与 Tauri 后端集成，实现状态的持久化存储
- **其他插件**：为其他插件提供状态管理能力

## 使用场景

- 需要访问用户登录状态的组件
- 需要显示或控制应用加载状态的组件
- 需要根据登录状态进行权限控制的组件
- 需要实现自动登录功能的应用启动流程
