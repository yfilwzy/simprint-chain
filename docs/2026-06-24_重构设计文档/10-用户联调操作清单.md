> 本清单是剩余四步 GUI 手动操作，用户在客户端点几下即可完成，每步附验证命令。

## 前置：启动客户端

```bash
cd C:\Users\ADMIN\simprint-build-src
pnpm install          # 重新装依赖（已清理 node_modules）
pnpm exec tauri dev   # 启动开发模式
```
等待编译完成（首次约 2-3 分钟），客户端窗口出现。

## 步骤 A：注册新账号

**操作**：客户端主窗口 → 注册 → 填邮箱+密码 → 提交

**验证 DB 落库**（开新终端）：
```bash
ssh simprint-server "docker exec simprint-postgres psql -U simprint_user -d simprint_db -c \"SELECT id, email, is_first_login, created_at FROM users ORDER BY created_at DESC LIMIT 5;\""
```
**预期**：返回一行，含你刚注册的邮箱，is_first_login=true。

**查服务器日志**：
```bash
ssh simprint-server "docker logs --tail 20 simprint-server 2>&1 | grep -i register"
```

## 步骤 B：账密登录

**操作**：退出（若已登录）→ 用刚注册的邮箱+密码登录

**验证 token 持久化**（客户端日志）：
```bash
# 客户端日志（应看到登录成功，无 decrypt failed）
tail -30 "C:/Users/ADMIN/AppData/Local/Simprint/logs/simprint_app.log" | findstr /i "login token session"
```

**验证服务器 sessions 表**：
```bash
ssh simprint-server "docker exec simprint-postgres psql -U simprint_user -d simprint_db -c \"SELECT s.user_id, u.email, s.created_at FROM sessions s JOIN users u ON s.user_id=u.id ORDER BY s.created_at DESC LIMIT 5;\""
```
**预期**：返回一行，含你的邮箱，对应一条 session。

**若失败排查**：
```bash
ssh simprint-server "docker logs --tail 50 simprint-server 2>&1 | grep -iE 'login|decrypt|api_secret|error'"
```
对照 `docs/2026-06-24_重构设计文档/08-加密协议实测规格.md` 核 padding/nonce。

## 步骤 C：创建环境

**操作**：客户端 → 新建浏览器环境 → 填名称/配置 → 保存

**验证 DB 落库**：
```bash
ssh simprint-server "docker exec simprint-postgres psql -U simprint_user -d simprint_db -c \"SELECT id, name, created_at FROM environments ORDER BY created_at DESC LIMIT 5;\""
```
**预期**：返回你刚创建的环境记录。

## 步骤 D：退出重登（remember 模式）

**操作**：退出客户端 → 重新启动 → 应自动登录（remember 模式）

**验证 remember**：
- 启动后**不需重新输密码**直接进入主窗口
- 客户端日志：
```bash
tail -20 "C:/Users/ADMIN/AppData/Local/Simprint/logs/simprint_app.log" | findstr /i "remember refresh"
```

**验证 keyring**（Windows 凭据管理器）：
- 控制面板 → 凭据管理器 → Windows 凭据 → 应有 `remembered-session.simprint`

## 联调成功的标志

四步全部：
- A 注册 → DB users 表有记录
- B 登录 → sessions 表有记录 + token 持久化
- C 创建环境 → environments 表有记录
- D 重登 → 自动登录无需密码

全部通过 = 隐私自托管闭环完整跑通。

## 故障排查速查

| 症状 | 查 |
|------|-----|
| 注册失败 | `docker logs simprint-server \| grep register` |
| 登录 decrypt failed | 对照 08 文档核 RSA padding |
| 401 unauthorized | sessions 表 + token 刷新 |
| 创建环境无响应 | local-api 路径 + docker logs |
| remember 不自动登录 | keyring + refresh_token |

服务器日志实时查（已配免密）：
```bash
ssh simprint-server "docker logs -f --tail 50 simprint-server"
```
